Practice 4

• Create a Registration/signup form where in following fields are mandatory, Additionally you can add more fields to explore more for e.g. – Gender, Nationality and so on
1) Username
2) Email
3) Password
4) Confirm Password
5) Signup/Register button
Form Validations should be applicable for all the fields if you leave any field blank or you enter invalid data. Once you click on Signup/Register button all the entered data should be visible either on console or on to the same page in the form of User JSON Object.
